0. 环境指纹 / 请求头诊断

网页可采集浏览器暴露的 UA、UA-CH、屏幕、时区、WebGL、canvas hash，以及服务端实际看到的请求头。纯线上浏览器不能直接读取本机 Codex/Claude CLI 配置、进程或遥测缓存；CLI 信息只有在 CLI/本地 companion 主动请求本站，或你手动导入诊断数据时才会出现。

0b. CLI / Proxy 捕获

三种来源：① CLI 直接请求 /api/fingerprint；② CLI 流量走 /proxy?target=...；③ 本地 companion 上传 /api/cli-report。捕获结果仅保存在当前服务内存里，重启即清空。默认脱敏；CTF 原文模式设置 CAPTURE_REDACT=false。

Companion 示例：node scripts/cli-companion.mjs --endpoint http://127.0.0.1:8787/api/cli-report；原文上传加 --no-redact
Proxy 示例：把目标请求发到 http://127.0.0.1:8787/proxy?target=https://example.test/path，服务会转发并记录请求/响应摘要。
